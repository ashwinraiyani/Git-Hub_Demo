• Initializing Git Respository : git init
  • Adding Remote Origin : git remote add origin https://github.com/awesomeinfosys/demo.git
  • Status of Git : git status
  • Adding All Files Created and Modified to Git : git add .
  • Adding Specific file to Git : git add filename
  • Commiting Code to Git : git commit -m "Commit Name"
  • Check Branches : git branch
  • Checking All Logs : git log
  • Checking Out Old Commit : git checkout commitID
  • Checking Master and Last Commited Code Active : git master
  • Resetting To Other Commit Active : git reset commitID
  • Revert the Changes : git checkout -- .
  • Create New Branch : git checkout -b branch-name
  • Checkout Master : git checkout master
  • Merge Branch To Master : git merge branch-name
  • Delete Branch : git branch -D branch-name
